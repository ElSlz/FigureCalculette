# calculette
Program to calculate arae and perimeter for different types of shapes.
Parameters:
	? : help
	-o: operation types
	-s: shape
	-r: inner radius
	-R: radius
	-b: base
	-w: width
	-h: height
	-a: side
	-c: side3
	
Shapes:
1: disc
2: rectangle
3: square
4: triangle
5: triangleisocele
6: trianglerectangle
7: ring

Operations:
1: area
2: perimeter

For example, passing arguments: -s 7 -o 2 -R 5 -r 3

Without any argument, the program will prompt instructions.
